# Module Pattern Bootstrap
An bootstrap that uses the Module design pattern.

## Installation

Clone into your project.
```shell
git clone git@github.com:guilhermepontes/module-pattern-bootstrap.git
```

## Usage
```javascript
MyApp.init();
```
